# Obsidian 学习笔记项目

这是我的个人学习笔记项目，使用 Obsidian 进行知识管理和学习记录。

## 📚 项目简介

这是一个基于 Obsidian 的知识管理仓库，包含我在学习过程中的笔记、总结和思考。Obsidian 是一个强大的知识管理工具，支持双向链接、图谱视图和 Markdown 编辑。

## 🗂️ 目录结构

```
obsidian/

│   ├── Python/

├── 系统设计/          # 系统设计学习笔记

└── 项目总结/          # 项目经验总结
```

## ✨ 主要特点

- **双向链接**: 笔记之间可以相互引用，形成知识网络
- **图谱视图**: 可视化展示笔记之间的关联关系
- **Markdown 支持**: 使用标准 Markdown 语法
- **本地存储**: 所有笔记都保存在本地，数据安全可控
- **插件生态**: 丰富的插件系统，可扩展功能

## 🚀 快速开始

### 1. 安装 Obsidian
- 官网下载: https://obsidian.md/
- 支持 Windows、macOS、Linux、iOS、Android

### 2. 打开本仓库
```bash
# 克隆仓库
git clone https://github.com/songhuiTan/obsidian.git

# 使用 Obsidian 打开文件夹
```

### 3. 开始使用
- 创建新笔记: `Ctrl/Cmd + N`
- 搜索笔记: `Ctrl/Cmd + O`
- 查看图谱: `Ctrl/Cmd + G`

## 📝 笔记规范

### 文件命名
- 使用英文或拼音，避免特殊字符
- 使用连字符分隔单词，如: `python-basics.md`
- 保持文件名简洁明了

### 内容结构
```markdown
# 标题

## 概述
简要介绍主题内容

## 主要内容
- 知识点1
- 知识点2
- 示例代码

## 总结
学习心得和总结

## 相关链接
- [[相关笔记1]]
- [[相关笔记2]]
```

## 🔗 双向链接使用

在 Obsidian 中，使用双方括号创建链接：
- `[[笔记名称]]` - 链接到其他笔记
- `[[笔记名称#标题]]` - 链接到笔记的特定标题
- `[[笔记名称|显示文本]]` - 自定义显示文本
