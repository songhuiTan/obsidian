### 新手路径
1. 先阅读 [[OpenClaw 新手必装 10 Skills 完整教程]]
2. 按顺序安装技能，每装一个就测试
3. 遇到问题查看 [[技能文档]] 中的具体说明
4. 完成任务后记录到 [[复盘日记]]

### 日常使用
1. 遇到问题先搜索 [[OpenClaw 学习索引]]
2. 查找对应技能的详细文档
3. 参考过往复盘中的解决方案
